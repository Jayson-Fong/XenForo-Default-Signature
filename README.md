# XenForo-Default-Signature
Set a default signature applied to newly registered members.

## Variables
- **{username}** - User's Username
- **{user_id}** - User's ID
- **{email}** - User's Email
- **{gender}** - User's Gender
- **{timezone}** - User's Timezone
- **{customField.***customFieldId***}** - Custom Field Data
